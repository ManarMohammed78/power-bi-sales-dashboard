# Business Performance Dashboard — Power BI

## Overview
A multi-page Power BI dashboard built on a star-schema sales data model (Customer, City, 
Date, Employee, Stock Item, and Sales fact tables), tracking key business KPIs.

## Tools
Power BI

## Key Metrics
- Total Sales: $19.92M
- Total Profit: $9.92M
- Total Cost: $9.96M
- Sales from 10–100 unit orders: $13.04M

## Features
- Pie and donut charts breaking down sales and profit by State/Province
- Line chart of cost trends by Buying Group
- Interactive pivot table and slicers for drill-down
- Decomposition tree and key-drivers visual for root-cause exploration
